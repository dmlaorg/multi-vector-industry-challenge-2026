# MULTI Vector Industry Challenge

This is a sandbox prepared for the MULTI 2026 challenge. Once you entered the sandbox, you can play with our DMLA solution that is the basis for our poster contribution: "Contractual Type-Square pattern and its lack of necessity in DMLA".

You have two options to enter the sandbox, via the Microsoft Dev Containers VS Code extension or GitHub codespaces.

## Microsoft Dev Containers

Install the official dev-containers extension from [VS Code marketplace](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers). 

With the plugin installed, VS Code will prompt the user to reload the sandbox in the dev container, or the `Dev Containers: Reopen in Container` can be used to load the sandbox within the container.

## GitHub Codespaces

Accessible directly on the cloud in a web view of GitHub via the upper-right green ` <> Code` menu item - choose `Codespaces` instead of `Local`.

Alternatively you can can use the remote access to Codespaces from your local VS Code installation via the official extension [VS Code marketplace](https://marketplace.visualstudio.com/items?itemName=GitHub.codespaces).
